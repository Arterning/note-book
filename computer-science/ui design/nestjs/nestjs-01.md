# install

```shell
npm i -g @nestjs/cli

nest new

npm run start

# in dev mode
npm run start:dev

#
nest generate controller
nest g co

nest generate service name
nest g s name

nest g module name


#generate dto class
nest g class name/dto/create-name.dto --no-spec
```
