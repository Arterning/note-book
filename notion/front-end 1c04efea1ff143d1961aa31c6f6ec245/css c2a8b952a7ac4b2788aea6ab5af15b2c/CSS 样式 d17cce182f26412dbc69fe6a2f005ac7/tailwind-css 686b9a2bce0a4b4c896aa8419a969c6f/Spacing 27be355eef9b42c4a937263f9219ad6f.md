# Spacing

间距
p-0 p-1..
pt=padding-top
pl=padding-left

px 表示水平方向padding包括padding-left padding-right
py 表示垂直方向
p-1 表示所有方向

默认p-1是4px
如果需要修改1到4的映射关系，可以在tailwind.config.js文件中配置

外边距同理，m-开头

space-x-4 表示flex-item在水平方向的间距是4

```css
<div class="flex space-x-4 ...">
  <div>01</div>
  <div>02</div>
  <div>03</div>
</div>
```
